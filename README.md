# CNN for CIFAR10 - Image Classification (PyTorch)

## Overview
Classifies images into 10 categories using a Convolutional Neural Network built in PyTorch.

## Dataset
- CIFAR10 — 60,000 images
- 10 classes: airplane, car, bird, cat, deer, dog, frog, horse, ship, truck

## Tech Stack
- Python, PyTorch, Torchvision
- Streamlit

## Architecture
- 3 Conv layers (32, 64, 128 filters)
- 2 Fully Connected layers
- MaxPooling + ReLU activations

## Results
- Test Accuracy: 74.71%

## How to Run
pip install -r requirements.txt
streamlit run app.py
